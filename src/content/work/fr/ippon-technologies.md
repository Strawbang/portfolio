### Modernisation Legacy chez Ippon Technologies

<p>En tant qu'Ingénieur Développeur Full-Stack chez Ippon Technologies depuis novembre 2025, je pilote des initiatives R&D internes pour accélérer la modernisation des systèmes legacy de clients stratégiques.</p>

<p>J'ai développé un CLI de modernisation piloté par l'IA en Rust, exploitant Tree-sitter pour le parsing AST et le RAG pour la documentation automatisée de code legacy — permettant aux équipes de comprendre et migrer des bases de code complexes plus rapidement.</p>

<p>J'ai mis en place des workflows agentiques via le Model Context Protocol (MCP), créant des agents de test autonomes et des assistants de refactoring qui interagissent directement avec les IDEs et les codebases pour automatiser les tâches de modernisation répétitives.</p>

<p>Ces initiatives ont significativement amélioré l'expérience développeur, en réduisant le temps d'onboarding sur les projets legacy grâce à des capacités interactives de "Chat with Code" pour les ingénieurs rejoignant des systèmes complexes.</p>
